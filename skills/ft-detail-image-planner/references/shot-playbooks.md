# Shot Playbooks

## A) Apparel (服装)
1. Hero on-model full body (front)
2. Hero on-model side/back
3. Fabric texture macro
4. Feature close-up (zipper/cuff/stitching)
5. Size guide graphic with model measurements
6. Styling scene (workout/street/office)
7. Color variants flat lay
8. Care instructions visual

## B) Footwear / Bags / Accessories
1. Hero 45-degree angle
2. On-person usage shot
3. Internal structure shot
4. Material close-up
5. Capacity / compartment map
6. Size comparison with common object
7. Lifestyle context

## C) Home / Kitchen / General Objects
1. Clean hero product shot
2. In-use scene
3. Dimensions annotated
4. Key part close-up
5. Before/after or problem-solution frame
6. Package contents
7. Installation steps (if needed)

## D) Tools / Electronics
1. Hero + key function icon
2. Interface/controls close-up
3. Performance proof visual (only factual)
4. Compatibility matrix
5. Safety or certification callout
6. Accessory list and package include
7. Workflow scene

## E) 1688 Wholesale Variant
- Add carton info frame (MOQ, pack qty, carton size)
- Add factory capability frame (OEM/ODM, lead time)
- Add bulk order CTA frame
