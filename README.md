+# Foreign Trade Skills Pack（外贸电商技能包）
+
+本技能包面向跨境与内外贸电商团队，覆盖“从素材到上架”的核心链路：
+- 图片驱动文案生成
+- 规格/材质结构化整理
+- 详情页图片策划
+- 多平台风格本地化
+- 合规与 SEO 发布前审查
+
+---
+
+## 1) `ft-product-copy-from-image`
+
+### 适用场景
+当你只有商品图片、零散聊天记录、简短工厂备注，缺少完整文案时使用。
+
+### 核心功能
+- 从图片中拆分**可见事实**与**推断信息**，避免把猜测当事实。
+- 自动生成平台化文案模块：
+  - SEO 标题（关键词优先）
+  - 5 点卖点（特性 -> 好处 -> 场景）
+  - 短描述/长描述
+- 自动输出“向供应商补数清单”（尺寸、材质、认证、装箱信息等）。
+- 增加“风险措辞提醒”，规避绝对化、无证据承诺类表述。
+- 在外贸语境不明确时可双语输出（中/英）。
+
+### 输入与输出
+- 输入：商品图、卖家备注、目标平台（可选）
+- 输出：一份可直接复用的 Product Copy Package（事实表 + 标题 + 卖点 + 描述 + 缺失字段）
+
+### 价值
+- 把“看图写文案”流程标准化，降低改稿轮次。
+- 显著减少后续因参数不明导致的上架返工。
+

+## 2) `ft-spec-material-table`
+
+### 适用场景
+当规格信息分散在聊天记录、图片文字、PDF 工厂单中，需要做成标准参数表时使用。
+
+### 核心功能
+- 将非结构化信息统一映射为标准规格字段（SKU、材质、尺寸、重量、MOQ、交期等）。
+- 对每个字段做状态标记：
+  - Provided（已提供）
+  - Estimated（估算）
+  - Missing（缺失）
+- 生成双层表格：
+  - 买家可读规格表（简洁）
+  - 内部 QC 校验表（含来源与风险）
+- 附带“供应商确认问题清单”，优先补齐退换货高风险字段。
+
+### 输入与输出
+- 输入：任意来源参数（文档、文本、图片识别结果等）
+- 输出：规格表 + 材质构成表 + 内部 QC 表 + 追问问题清单
+
+### 价值
+- 统一产品数据口径，减少平台间字段冲突。
+- 提前暴露风险参数（尺寸公差、脆弱件、涂层等）。
+

+## 3) `ft-detail-image-planner`
+
+### 适用场景
+当你要制作详情页图（拍摄或 AI 生成）但没有明确镜头脚本时使用。
+
+### 核心功能
+- 按品类自动切换图片叙事策略：服装/配饰/家居/工具电子等。
+- 产出 6-10 张详情图分镜，每张图标注：
+  - 目标（建立信任、证明功能、促转化）
+  - 必备元素（模特、道具、尺寸标注、结构拆解等）
+  - 覆盖文案（短句、利益点导向）
+- 内置玩法：
+  - 服装必须有模特上身 + 面料特写 + 尺码引导图
+  - 物品必须有比例尺参考 + 关键细节特写 + 使用场景
+- 提供 Amazon 与 淘宝/1688 的风格适配建议（简洁 vs 注释丰富）。
+
+### 输入与输出
+- 输入：商品类型、卖点、目标平台、素材条件（是否有模特/场景）
+- 输出：主图方案 + 详情分镜 + 文案叠字草案 + 修图清单 + A/B 测试思路
+
+### 价值
+- 避免“凭感觉出图”，提高图片制作一致性与转化效率。
+- 明确摄影/设计团队协作分工，减少反复沟通。
+

+## 4) `ft-marketplace-listing-localization`
+
+### 适用场景
+同一个商品要同时发布到 Amazon、淘宝、1688，且希望“事实一致、表达风格不同”时使用。
+
+### 核心功能
+- 先冻结统一事实底稿（参数、卖点、交易条件）。
+- 再按渠道改写表达方式：
+  - Amazon：搜索与信息清晰优先
+  - 淘宝：情绪化种草与视觉叙事优先
+  - 1688：工厂能力、MOQ、交期、定制能力优先
+- 提供一致性差异检查，标出潜在冲突字段。
+
+### 输入与输出
+- 输入：统一产品事实、目标渠道
+- 输出：三端版本文案 + 一致性对照结果
+
+### 价值
+- 保证多平台协同运营时“说法不打架”。
+- 同时兼顾 B2C 与 B2B 受众沟通效率。
+

+## 5) `ft-listing-compliance-seo`
+
+### 适用场景
+文案、详情图文、参数表已完成，准备发布前做最后风控与 SEO 体检时使用。
+
+### 核心功能
+- 扫描所有文案与图文叠字，识别高风险声明：
+  - 绝对化承诺
+  - 无依据认证宣称
+  - 医疗功效暗示
+  - 物流/效果过度保证
+- 将语句分级：Safe / Needs evidence / High-risk。
+- 生成“低偏差改写建议”：尽量保留卖点但降低合规风险。
+- 执行 SEO 检查：主关键词覆盖、属性词覆盖、堆砌风险与冗余。
+
+### 输入与输出
+- 输入：完整 listing 文案与图片叠字内容
+- 输出：风险清单 + 安全改写 + 关键词覆盖报告 + 发布前 checklist
+
+### 价值
+- 降低审核驳回、投诉和售后纠纷概率。
+- 在安全前提下保持搜索可见度与转化效率。
